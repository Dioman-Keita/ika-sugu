## ⚡ Features  

### 🛍️ Storefront  
- Dynamic product catalog  
- Advanced multi-filter system  
- Real-time product variants (SKU tracking)  
- Internationalization (🇫🇷 French / 🇬🇧 English)  
- Dark mode support  

---

### 💳 Checkout & Payments  
- Full checkout flow (built from scratch)  
- Stripe integration  
- Local testing with Stripe CLI  

---

### 🛠️ Admin & Backoffice  
- Full admin dashboard (not just UI — real impact on storefront)  
- Product creation with **image upload (up to 6 per variant)**  
- TVA (tax) management system  
- End-to-end product management  

---

### 🧠 Data & Logic  
- Prisma ORM for database management  
- Clean handling of product variants and SKUs  
- Server actions for backend operations  

---

## 🧱 Tech Stack  

- **Frontend:** Next.js, React, TailwindCSS  
- **State Management:** TanStack Query  
- **Backend:** Next.js API / Server Actions  
- **Database:** PostgreSQL + Prisma  
- **Payments:** Stripe  
- **Deployment:** Vercel  
- **Infra:** (optional: Cloudflare if used)  

---

## 🧠 Architecture  

- Separation between UI, business logic, and data layer  
- Scalable product & variant modeling (SKU-based)  
- Admin system directly impacting frontend state  
- Hybrid data fetching (server + client with TanStack Query)
